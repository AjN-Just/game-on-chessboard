## Description
Game on Chessboard est un projet qui implémente un jeu / une simulation sur un échiquier. L'objectif est de fournir une base claire pour :
- la logique du jeu (mouvements, règles, détection de fin de partie),
- une interface utilisateur (web / CLI / desktop),
- des tests et exemples de parties.

## Fonctionnalités
- Initialisation d'un échiquier standard
- Déplacement des pièces selon les règles
- Validation des coups (simples / avancés)
- Sauvegarde / chargement d'une partie (optionnel)
- Mode solo (IA basique) et mode multijoueur (optionnel)

## Installation

1. Prérequis
   - Installez [Node.js](https://nodejs.org/) (si projet JS/TS) ou Python/Java selon le cas.
   - Autres prérequis : Git, Docker (si utilisé), etc.

2. Cloner le dépôt
```bash
git clone https://github.com/AjN-Just/game-on-chessboard.git
cd game-on-chessboard
```

3. Exemple d'installation (Node.js)
```bash

# ou avec yarn
```
